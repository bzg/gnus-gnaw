#+title: gnus-gnaw.el --- Highlight BONE reports in Gnus

[[https://intver.org/][https://img.shields.io/badge/versioning-intver.org-blue.svg?style=for-the-badge]]

*This library is not actively maintained, it is shared as a proof of
concept.  If you want to maintain and develop it, please [[mailto:bzg@bzg.fr][contact me]].*

=gnus-gnaw= highlights Gnus summary lines whose =Message-Id= matches an
open [[https://codeberg.org/bzg/bone][BONE]] report. It reads the same =reports.json= sources as [[https://codeberg.org/bzg/gnaw][gnaw]].

* Requirements

- The [[https://codeberg.org/bzg/gnaw.el][gnaw.el]] library, which provides the shared data layer
  (configuration, report cache and =state.edn= handling).  Once
  =gnus-gnaw= is on MELPA this dependency will be pulled in
  automatically; until then, see /Installation/ below.
- =~/.config/gnaw/config.edn= (an EDN map with =:my-addresses= and
  =:sources= keys, shared with gnaw)
- =gnus-gnaw= supports =reports.json= bone-format =0.9.3=.

* Installation

=gnus-gnaw= is not on MELPA yet.  On Emacs 30 or later, install it from
git with the built-in =package-vc=.  Install [[https://codeberg.org/bzg/gnaw.el][gnaw.el]] *first*, since it
is its dependency and is not in any package archive:

#+begin_src emacs-lisp
(use-package gnaw
  :vc (:url "https://codeberg.org/bzg/gnaw.el" :rev :newest))

(use-package gnus-gnaw
  :vc (:url "https://codeberg.org/bzg/gnus-gnaw" :rev :newest))
#+end_src

Or interactively, in order:

: M-x package-vc-install RET https://codeberg.org/bzg/gnaw.el RET
: M-x package-vc-install RET https://codeberg.org/bzg/gnus-gnaw RET

Update later with =M-x package-vc-upgrade=.

=gnus-gnaw= does not require the [[https://codeberg.org/bzg/gnaw][gnaw]] CLI: it reads =config.edn= and
=reports.json= files directly.  You can use it without installing the
=gnaw= binary, as long as the configuration file and the report sources
it references are available.

* Commands

- =M-x gnus-gnaw= — Apply overlays to the current summary buffer.
- =M-x gnus-gnaw-highlight= — Highlight without limiting the summary.
- =M-x gnus-gnaw-topic= — Filter reports by selected topic.
- =M-x gnus-gnaw-clear= — Remove all =gnus-gnaw= overlays.

** Local marks

The following commands toggle gnaw's local marks (stored in
=~/.config/gnaw/state.edn= and shared with the [[https://codeberg.org/bzg/gnaw][gnaw]] CLI).

- =M-x gnus-gnaw-mark-sticky= — toggle the sticky mark (keep the report visible).
- =M-x gnus-gnaw-mark-skip= — toggle the skip mark (hide the report).

The annotation gains a leading mark column: =*= for sticky, =_= for
skip, space otherwise.
