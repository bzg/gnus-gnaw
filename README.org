#+title: gnus-bone.el --- Highlight BARK reports in Gnus

[[https://intver.org/][https://img.shields.io/badge/versioning-intver.org-blue.svg?style=for-the-badge]]

=gnus-bone= highlights Gnus summary lines whose =Message-Id= matches an
open [[https://codeberg.org/bzg/bark][BARK]] report. It reads the same =reports.json= sources as [[https://codeberg.org/bzg/bone][bone]].

* Requirements

- =~/.config/bone/config.edn= (an EDN map with =:addresses= and
  =:sources= keys, shared with bone)
- =gnus-bone= supports =reports.json= bark-format =0.9.1=.

=gnus-bone= does not depend on [[https://codeberg.org/bzg/bone][bone]]: it reads =config.edn= and
=reports.json= files directly.  You can use it without installing bone,
as long as the configuration file and the report sources it references
are available.

* Commands

- =M-x gnus-bone= — Apply overlays to the current summary buffer.
- =M-x gnus-bone-clear= — Remove all =gnus-bone= overlays.
- =M-x gnus-bone-topic= — Filter reports by selected topic.
